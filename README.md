## 🚩 已经实现的功能
* 暴力自动重连，解决iwara土豆服务器总是无响应问题
* 登录/查看个人信息
* 浏览订阅更新列表
* 播放视频
* 查看图片
* 查看评论
* 点赞
* 关注

## 🎨 主要技术栈
* MVVM 架构
* Jetpack Compose (构建UI)
* Kotlin Coroutine (协程)
* Okhttp + Jsoup (解析网页)
* Retrofit (访问Restful API)
* EventBus (事件总线)
* Hilt (依赖注入)
* Paging3 (分页加载)
